# TestPilot AI

LangGraph-powered AI QA copilot that assists exploratory testing by generating module analyses, risk-based checklists, and execution-ready manual test cases from application workflows.

## Tech Stack

- Python
- FastAPI
- LangGraph
- Google Gemini
- Pydantic

## Features

- Module identification
- Exploratory checklist generation
- Manual test case generation
- Support for observed user steps
- Multi-agent architecture
